# Python汽车数据可视化系统

## 核心功能
- 用户登录注册
- 条件选车
- 销量榜单
- 差评榜单
- 汽车数据可视化分析

## 适用场景
- 汽车行业数据分析
- 汽车市场研究
- 消费者购车决策支持

## 技术特点
- 采用Django、MySQL、Vue技术栈
- 提供用户友好的数据可视化界面
- 支持多种数据源接入与处理
